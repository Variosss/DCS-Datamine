# DCS-Datamine
Repository updated after every DCS update to keep track of all the changes.<br>
I don't own all the modules so if somebody owns more make a pull requests so I can include them.<br>
Inspired by gszabi99's war thunder datamine repository.
# How to help me
Open DCS folder in CMD and type
```
git init
git remote add origin https://github.com/Variosss/DCS-Datamine
git pull origin main
``` 
After this just commit the changes.
